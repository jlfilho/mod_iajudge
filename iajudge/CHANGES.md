# Changelog

## 0.2.1

- Bumped plugin version after JS, settings, and language changes.
- Added `db/upgrade.php` skeleton for future Moodle upgrade steps.
- Replaced browser `alert()` validation with an inline Moodle-styled error box.
- Aligned package documentation with the Moodle 5.2 target.

## 0.1.0

- Initial release of the `mod_iajudge` activity module.
- Teacher form with intro editor, rubric prompt, allowed languages, and attempt limit.
- Student submission flow with asynchronous AI evaluation.
- Gradebook integration for scored submissions.
- External provider support scaffolding for OpenAI, Anthropic, Gemini, and Ollama.
- Ad-hoc task processing for background grading.
